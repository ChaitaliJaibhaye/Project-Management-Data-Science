Project Management for Data-Driven Healthcare
Predictive Analytics Case Study

Project Overview-
This repository contains an academic case study on using predictive analytics to reduce 30-day hospital readmissions. The project highlights how effective project management enables data science teams to align technical work with clinical needs, stakeholder expectations, and measurable outcomes.

Role of Project Management in Data Science-

• Defined clear goals, scope, and success metrics
• Coordinated cross-functional teams including data scientists, clinicians, and IT staff
• Used Work Breakdown Structure to manage analytics tasks
• Applied Agile methods for model iteration and feedback
• Managed data privacy, compliance, and system integration
• Supported user adoption through training and change management

Key Outcome-
The project achieved a 12 percent reduction in 30-day readmissions, demonstrating that strong project management is essential for turning data science models into real-world impact.
